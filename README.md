# Memo

## App for saving your most valuable memories

**Made with:**

-   React Native
-   React Native Async Storage

**Install Expo**

-   App is deployed using Expo CLI
-   Install Expo CLI globally on your machine

**Install Node modules**

```
	npm install
```

**Run App locally:**

```
	npm run ios
```
